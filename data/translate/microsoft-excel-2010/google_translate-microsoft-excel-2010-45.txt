Filtros de texto avançados Os filtros de texto avançados são úteis para exibir informações mais específicas, como células que contêm um determinado número de caracteres ou dados que não contêm uma palavra específica. Neste exemplo, usaremos filtros de texto avançados para ocultar qualquer equipamento relacionado a câmeras, incluindo câmeras digitais, filmadoras e muito mais. Para isso, siga estas instruções: Etapa 1: Selecione a guia Dados e clique no comando Filtrar. Passo 2: Agora, clique na seta para baixo na coluna de texto que você vai filtrar. Neste exemplo, Detalhes da equipe. Etapa 3: selecione a opção Filtros de texto para abrir o menu de opções de filtro avançado. Passo 4: Escolha um filtro. Neste exemplo, escolheremos Não contém . Passo 5 – A caixa de diálogo Custom AutoFilter aparecerá. Lá, digite o texto à direita do seu filtro. Neste exemplo, vamos inserir a palavra &quot;cam&quot; para ver os dados que não contêm essas letras. Isso exclui qualquer equipamento relacionado à câmera, como câmeras digitais, câmeras de vídeo, bolsas para câmeras e impressoras de câmeras digitais. Passo 6: Clique em OK. Os dados são filtrados com base no argumento escolhido e no texto que você especificou.
