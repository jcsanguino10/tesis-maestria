Alterando Dados em Tabelas Dinâmicas Uma das melhores coisas sobre tabelas dinâmicas é que elas permitem que você altere os dados para responder a várias perguntas e até mesmo experimentar os dados. Em nosso exemplo, usamos a tabela dinâmica para responder à pergunta: qual é o total vendido por cada vendedor? No entanto, agora gostaríamos de responder a uma nova pergunta, por exemplo, qual é o total vendido em cada mês? Podemos fazer isso apenas alterando os rótulos das linhas. Para fazer isso, siga estas etapas: Etapa 1: Arraste os campos existentes para fora da área Rótulos de linha e eles desaparecerão. Passo 2: Arraste um novo campo da Lista de Campos para a área Rótulos de Linha. Neste exemplo, estamos usando o campo Mês. Etapa 3 – A tabela dinâmica se ajusta para exibir os novos dados. Neste exemplo, ele agora nos mostra a venda total em cada mês.
