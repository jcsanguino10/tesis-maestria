Como criar e salvar um arquivo? O primeiro passo que você deve seguir para fazer seu orçamento no Excel 2010 é criar uma nova pasta de trabalho. Criar uma nova pasta de trabalho ou arquivo Etapa 1: Depois de abrir o Excel 2010, clique na guia Arquivo. Você o reconhecerá facilmente porque é o único cuja cor é verde. Etapa 2: você verá uma lista de botões na guia Arquivo. Lá, clique em Novo. Passo 3: Agora, aperte o botão Blank Book localizado dentro dos templates disponíveis para criar um novo arquivo. Passo 4: Para finalizar, clique no botão Criar, localizado no canto inferior direito. Etapa 5: a primeira coisa que você verá ao criar uma pasta de trabalho em branco será uma planilha. Salvar um arquivo ou livro Etapa 1: Clique na guia Arquivo. Você vai identificá-lo porque é o único verde. Passo 2: Agora, localize e clique no comando Salvar. Passo 3: Você verá uma caixa de diálogo. Lá, encontre o local em seu computador onde seu arquivo será salvo. Passo 4: Na caixa Nome do arquivo, digite o título que irá identificá-lo. Passo 5: Para finalizar, clique no botão Salvar.
