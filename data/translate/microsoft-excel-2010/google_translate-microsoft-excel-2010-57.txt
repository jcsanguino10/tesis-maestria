Alterar o estilo do gráfico Outra opção que você tem para modificar a aparência de seus gráficos é alterar seu estilo. Vamos ver como é fácil fazer isso. Etapas para alterar o estilo de um gráfico: Etapa 1: selecione a guia Layout e clique na seta suspensa no grupo Estilos de layout para ver todos os disponíveis. Etapa 2: selecione o estilo que deseja para o seu gráfico. Como mover o gráfico? Em algumas ocasiões, você precisará mudar seus gráficos para uma planilha diferente. Este procedimento é muito simples. Preste muita atenção. Etapa 1: selecione a guia Layout e clique no gráfico que deseja mover. Em seguida, clique no comando Mover Gráfico. Passo 2: Selecione o local desejado para a mesa. Ou seja, você escolhe uma planilha existente selecionando a opção Objeto em, ou uma nova selecionando a opção Nova Planilha. Passo 3: Para finalizar, clique em OK. A tabela aparecerá na planilha selecionada na caixa de diálogo.
