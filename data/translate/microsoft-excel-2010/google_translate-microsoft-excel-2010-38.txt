Mostrar ou ocultar dados Ocultar grupos é uma boa ideia quando você tem muitas informações e precisa ver apenas o necessário. Vamos ver como fazê-lo. Etapa 1: clique no sinal de menos, também conhecido como símbolo Ocultar detalhes, para recolher o grupo. Etapa 2: clique no sinal de mais, também conhecido como símbolo Mostrar detalhes, para expandir o grupo novamente. Observe… Você também pode usar os comandos da guia Dados localizados no grupo Esquema para realizar este procedimento.
