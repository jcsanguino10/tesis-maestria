O que é e como criar uma tabela dinâmica? Uma tabela dinâmica é usada para resumir os dados em uma planilha. O melhor de tudo é que você pode alterá-lo de maneira rápida e fácil para visualizar os dados de uma maneira diferente, tornando-a uma ferramenta muito poderosa. Como criar uma tabela dinâmica? Partindo de uma planilha contendo estatísticas de vendas de uma empresa fictícia, suponha que queiramos responder à pergunta: quanto é o total de vendas de cada vendedor? Isso pode ser demorado porque cada um deles aparece em várias linhas, assim como sua venda mensal. Embora pudéssemos usar a função Subtotal, ainda teríamos muitos dados para analisar. Felizmente, uma tabela dinâmica pode fazer toda a matemática instantaneamente e resumir os dados de uma forma que não é apenas fácil de ler, mas também fácil de manipular. Para criar uma tabela dinâmica que responda à pergunta, siga estas etapas: Etapa 1: selecione a tabela ou as células, incluindo os cabeçalhos das colunas, que contêm os dados que você usará. Etapa 2: Na guia Inserir, clique no comando Tabela Dinâmica. Etapa 3: A caixa de diálogo Criar tabela dinâmica será exibida. Verifique se as configurações estão corretas e clique em OK. Passo 4: Uma tabela dinâmica em branco aparecerá no lado esquerdo e a lista de campos no lado direito.
