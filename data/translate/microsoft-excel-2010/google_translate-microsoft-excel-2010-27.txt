O que é uma função no Excel? Uma função é uma fórmula predefinida que realiza cálculos usando valores específicos em uma determinada ordem. Uma das principais vantagens é que eles economizam tempo porque você não precisa mais escrevê-los sozinho. As partes de uma função Cada função tem uma sintaxe, uma ordem específica que deve ser seguida para obter o resultado correto. A sintaxe básica para criar uma fórmula com uma função é: Insira um sinal de igual (=) . Selecione uma função ( SUM , por exemplo, é o nome da função para adição). Insira os argumentos da fórmula, ou seja, os dados que serão utilizados para fazer o cálculo. Para usar essas funções corretamente, você precisa saber quais são as partes de uma função e como criar argumentos para calcular valores e referências de células. Trabalhando com argumentos básicos Os argumentos devem ser colocados entre parênteses. Valores individuais ou referências de células, separados por dois pontos ou vírgulas. Observe como você deve fazer isso em cada caso: Dois pontos Crie uma referência a um intervalo de células. O exemplo a seguir calcula a média do intervalo de células E19 a E23. =MÉDIA(E19:E23) Vírgulas Separe valores individuais, referências de células e intervalos de células entre parênteses. Se houver mais de um argumento, você deve separar cada argumento por uma vírgula. =COUNT(C6:C14,C19:C23,C28) Este exemplo conta todas as células nos três argumentos entre parênteses.
