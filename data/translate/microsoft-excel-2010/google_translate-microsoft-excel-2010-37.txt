Agrupar os dados em uma planilha Agrupar os dados que você tem em uma planilha é útil para organizá-la. Desta forma, você poderá fazer análises rápidas usando o comando Subtotal e também decidir se deseja ocultar ou mostrar informações. Vamos ver do que se trata. O comando Subtotal pode ser usado para organizar sua planilha de várias maneiras. Ele usa funções comuns como SUM , COUNT , AVG e mais, para resumir os dados e colocá-los em um grupo. Para saber mais sobre funções, visite nossa lição Explorar funções básicas do Excel 2010. Agrupar dados com Subtotal Vamos supor que você vende camisetas e precisa saber quantas deve entregar por tamanho. Siga estas etapas: Etapa 1: classifique os dados de acordo com o que você deseja contar. Neste exemplo, vamos organizar a planilha por tamanho de camisa. Revise a página Classificar dados por critérios personalizados para obter mais informações. Passo 2: Selecione uma célula vazia que esteja logo acima dos dados que você vai contar. Em seguida, clique na guia Dados e localize o grupo Esquema. Passo 3: Clique no comando Subtotal. Você verá uma pequena janela com um aviso. Lá, aperte o botão OK. A caixa de diálogo Subtotais será exibida. Etapa 4: no campo Para cada alteração em, clique na coluna que contém os dados a serem contados. Neste caso, escolheremos a coluna Tamanho. Etapa 5: No campo Usar função, selecione uma das opções disponíveis na lista. Aqui, vamos usar a função COUNT para obter o número total de camisas por tamanho. Passo 6: Agora, no campo Adicionar subtotal a, selecione a coluna na qual a contagem aparecerá. Neste caso, vamos escolher a coluna T alla. Passo 7: Para finalizar, clique em OK. Importante... Você verá que cada tamanho de camiseta é colocado em seu próprio grupo e o subtotal da contagem é exibido abaixo de cada um.
