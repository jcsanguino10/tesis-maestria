Alterar a ordem dos dados e o layout de um gráfico Às vezes, quando você cria um gráfico, os dados podem não ser agrupados da maneira desejada. No entanto, você pode alterar a ordem dos dados a qualquer momento. Veja como fazer. Etapas para alterar os dados das linhas e colunas Etapa 1: Selecione o gráfico para o qual deseja alterar os dados. Localize a guia Layout e selecione o comando Alternar entre linhas e colunas. Passo 2: Se você olhar para o gráfico novamente, verá que os dados que você modificou aparecerão no gráfico. Etapas para alterar o layout do gráfico Você pode alterar o layout do gráfico a qualquer momento. Vamos ver como fazer isso de forma fácil e rápida. Etapa 1 – Selecione a guia Layout e clique na seta suspensa no grupo Layouts de gráfico para ver todos os disponíveis. Etapa 2: selecione o design que deseja aplicar ao seu gráfico. Tome nota… Alguns layouts incluem títulos de gráfico, eixos ou legendas. Para alterá-los, basta colocar o ponto de inserção no texto e começar a digitar.
