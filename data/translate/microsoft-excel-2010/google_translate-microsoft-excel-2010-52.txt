Gráficos em pastas de trabalho do Excel Excel podem conter uma grande quantidade de dados que geralmente são difíceis de interpretar. Por exemplo, onde estão os valores máximo e mínimo? Os números estão aumentando ou diminuindo? As respostas a perguntas como essas são mais claras quando os dados são representados por um gráfico. Tipos de gráficos: O Excel oferece vários tipos de gráficos que têm suas próprias vantagens. São eles: Coluna: Os gráficos de coluna são simples, mas versáteis. Eles podem ser usados para exibir quase qualquer tipo de dados e existem muitas variações, como 3-D, empilhado, pirâmide e outros. O exemplo mostrado é um gráfico de colunas agrupadas. Linha: Os gráficos de linha são ideais para mostrar tendências. Os pontos são conectados com linhas, então é fácil ver quais números estão aumentando ou diminuindo. Pizza: os gráficos de pizza permitem que você veja as proporções rapidamente. Cada valor é mostrado como uma fatia do bolo, portanto, é fácil ver qual valor representa a maior e a menor porcentagem. Barras: os gráficos de barras funcionam como os gráficos de colunas, mas em vez de colunas verticais, eles usam barras horizontais. Área: Os gráficos de área são muito semelhantes aos gráficos de linhas, exceto que a área abaixo das linhas é preenchida. Isso pode facilitar a leitura dos dados, embora em alguns casos o preenchimento possa ocultar informações. Superfície Os gráficos de superfície exibem dados em uma espécie de paisagem 3D. Eles funcionam melhor com grandes conjuntos de dados, permitindo que você veja exatamente onde os valores são mais altos e mais baixos.
