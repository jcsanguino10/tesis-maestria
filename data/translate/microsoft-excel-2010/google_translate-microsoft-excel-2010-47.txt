Usando filtros numéricos avançados Os filtros numéricos avançados permitem manipular dados numéricos de várias maneiras. Vamos ver do que se trata. Como usar filtros avançados por número? Neste exemplo, mostraremos apenas determinados tipos de equipamentos com base em seu código de registro. Para isso, siga estas instruções: Etapa 1: Selecione a guia Dados e clique no comando Filtrar. Passo 2 – Clique na seta para baixo na coluna de números que você deseja filtrar. Neste exemplo, Código . Etapa 3: escolha Filtros de número para abrir o menu de opções de filtro avançado. Etapa 4: selecione um filtro. Neste exemplo, vamos escolher Between para ver os códigos que existem entre os números que você especificar. Passo 5: Você verá a caixa de diálogo Custom AutoFilter aberta. Etapa 6: digite um número à direita de cada filtro. Neste exemplo, veremos códigos maiores ou iguais a 3000, mas menores ou iguais a 4000. Isso deve exibir códigos que estejam no intervalo de 3000-4000. Passo 7: Para finalizar pressione o botão OK. Os dados são filtrados com base no argumento e nos números que você especificou.
