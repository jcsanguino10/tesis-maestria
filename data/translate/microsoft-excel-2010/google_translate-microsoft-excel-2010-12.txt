Modificar a altura das células Se você deseja modificar a altura das linhas, deve realizar um procedimento muito simples. Vamos ver quais são os passos que você terá que seguir: Como modificar a altura da linha? Passo 1: Passe o mouse sobre as linhas que separam os cabeçalhos das linhas. Você verá que a cruz branca se transforma em uma seta dupla. Passo 2: Clique e, com o botão do mouse pressionado, arraste a linha para baixo ou para cima para aumentar ou diminuir sua altura. Passo 3: Solte o mouse quando atingir a altura desejada. Você verá que a largura da coluna mudou. Modifique a altura de uma linha com uma medida exata: você pode usar o comando Altura da Linha para ser mais exato com a medida do que com a altura de uma linha. Etapa 1: selecione as linhas cuja altura você deseja modificar e clique no comando Formatar na guia Início. Etapa 2: Agora, selecione a opção Altura da linha no menu suspenso. Passo 3 – A caixa de diálogo Altura da Linha será aberta. Lá, digite um número específico e clique no botão OK. Outra maneira de Ajustar Linhas é: Selecione a opção AutoAjustar Altura da Linha no mesmo menu suspenso. Automaticamente cada linha selecionada será ajustada ao texto que está nela.
