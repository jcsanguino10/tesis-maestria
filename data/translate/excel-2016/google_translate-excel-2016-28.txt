Gráficos dinâmicos no Excel 2016 Além das tabelas dinâmicas, o Excel permite criar gráficos dinâmicos para obter uma melhor perspectiva de suas informações. Você pode criar um gráfico dinâmico sem criar uma tabela dinâmica, basta seguir estes passos: Passo 1: Selecione uma célula da tabela ou dos dados que deseja ver representados no gráfico. Etapa 2: Na guia Inserir da Faixa de Opções, clique no comando Gráfico Dinâmico. Etapa 3: A caixa de diálogo Criar gráfico dinâmico será aberta. A partir daí, você pode escolher se deseja que o gráfico seja criado na mesma planilha da tabela ou em uma nova planilha. Você também poderá selecionar se a fonte de dados que usará é a selecionada ou se deseja usar uma fonte de dados externa e se deseja analisar várias tabelas ao mesmo tempo. Para este exemplo, usaremos uma tabela existente e criaremos o gráfico em uma nova planilha. Quando terminar, clique no botão OK. Passo 4: Será criada uma nova planilha onde ficará o gráfico dinâmico e um painel de onde você poderá controlá-lo. Como funciona um gráfico dinâmico: junto com o gráfico, um painel será aberto a partir do qual você poderá controlá-lo e selecionar como deseja classificar ou filtrar as informações nele contidas. À medida que você cria o gráfico dinâmico, o sistema também cria uma tabela dinâmica. Como excluir um gráfico dinâmico: Etapa 1: Para excluir um gráfico dinâmico, a primeira coisa que você precisa fazer é selecioná-lo. Etapa 2: em seguida, na faixa de opções, vá para a guia Analisar e clique no comando Ações. Passo 3: Será exibido um menu onde você deve clicar na opção Excluir. Passo 3: Um menu será exibido. Lá, clique na opção Apagar tudo. O sistema excluirá o gráfico imediatamente e você poderá criar um novo.
