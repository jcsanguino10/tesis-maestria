Exportando um documento no Excel 2016 Por padrão, o Excel salva pastas de trabalho no formato .xlsx, mas às vezes pode ser necessário usar outro tipo de formato, como PDF ou Excel 97-2003. É muito fácil exportar uma planilha do Excel para outros tipos de formato de arquivo, vamos ver como. Como exportar uma planilha para um arquivo PDF: Este tipo de formato é especialmente útil quando você precisa compartilhar o arquivo com alguém que não tem Excel para abrir o documento. Todos que receberem o arquivo poderão visualizá-lo, mas não editá-lo. Etapa 1: Clique na guia Arquivo para acessar a visualização Backstage. Etapa 2: no menu Backstage, clique em Exportar. Etapa 3: No painel Exportar, selecione a opção Criar documento PDF/XPS. Lá, clique no botão Criar documento PDF/XPS. Etapa 4: A caixa de diálogo Publicar como PDF ou XPS será aberta. Lá, selecione o local para onde deseja exportar a pasta de trabalho, digite o nome que o arquivo terá e clique no botão Publicar. Por padrão, o Excel exporta apenas planilhas ativas. Se você tiver um documento de várias planilhas que deseja exportar para um PDF, na caixa de diálogo Salvar como, clique no botão Opções. Uma caixa será aberta onde você deve selecionar a opção Pasta de trabalho inteira e clicar em OK. Exporte uma planilha para outro tipo de arquivo: o recurso Exportar do Excel 2016 também permite converter uma planilha para versões anteriores do Excel ou para um arquivo .CSV, se você precisar de uma versão de texto simples. Etapa 1: Na visualização Backstage, clique em Exportar. Etapa 2: no painel Exportar, clique na opção Alterar tipo de arquivo. Passo 3: Uma caixa será aberta com os diferentes tipos de formato em que você pode exportar um documento. Clique no tipo de formato que deseja selecionar e, em seguida, clique no botão Salvar como . Etapa 4: A caixa de diálogo Salvar como será aberta. Lá, selecione o local onde deseja salvar o documento, digite o nome que ele terá e, por fim, clique no botão Salvar.
