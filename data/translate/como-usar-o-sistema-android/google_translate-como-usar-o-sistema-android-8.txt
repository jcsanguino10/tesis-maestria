Restaurar o aparelho para as configurações de fábrica Você pode vender ou dar seu celular para um familiar sem se preocupar em apagar manualmente as informações que você salvou dados por dados. Você pode restaurar o dispositivo às configurações iniciais de fábrica seguindo estas etapas. Etapa 1: abra o aplicativo Configurações ou Configurações. Passo 2: No painel de configurações do seu dispositivo, toque na opção Backup e redefinição. Passo 3: Um menu será aberto onde você encontrará as opções para criar um backup do seu computador e restaurar o sistema. Lá, selecione a opção de redefinição de dados de fábrica. Passo 4: Você verá uma notificação dos dados que o sistema excluirá e as contas que serão fechadas. Depois de ler todas as informações, pressione o botão Redefinir dispositivo. Etapa 5: o sistema solicitará que você confirme que deseja fazer uma redefinição de fábrica. Toque no botão Limpar tudo. Etapa 6: O sistema será redefinido e todas as informações de armazenamento interno serão apagadas. Este processo é muito difícil de desfazer. É recomendável que você faça backup de suas informações antes de realizar uma redefinição de fábrica para evitar o risco de perder algo importante. Lembre-se que como todos os processos do Android, os nomes dos botões podem variar dependendo da versão do sistema operacional ou do equipamento que você possui.
