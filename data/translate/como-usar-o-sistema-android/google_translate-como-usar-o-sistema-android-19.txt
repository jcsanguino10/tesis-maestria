Navegadores para Android Google Chrome para Android: Google Chrome é o navegador padrão para Android. Para abrir o navegador, clique no ícone do Chrome. A interface do navegador consistirá em: 1. Barra de pesquisa: Neste espaço você pode escrever a URL ou endereço web do site que deseja acessar. Você também pode digitar o termo do tópico que deseja pesquisar online, por exemplo, educação ou tecnologia. 2. Botão Configurações: Este botão exibirá um menu de configurações. A partir daí, você poderá abrir uma nova guia, acessar suas páginas marcadas, acessar seu histórico de navegação e imprimir o que estiver vendo na Internet. 3. Favoritos: Este botão aparecerá apenas na página principal do navegador. Ele permitirá que você acesse as páginas da web que você marcou como favoritas. Você também encontrará esta opção no menu de configuração. 4. Abas recentes: Permite acessar as páginas que você visitou recentemente. Assim como os Favoritos, você verá o botão apenas na página inicial, mas poderá acessar esse recurso no menu de configurações do seu navegador. Navegador de Internet: Em alguns computadores, você encontrará um navegador chamado Internet. Este foi o navegador Android até a versão 4.1 do sistema operacional, chamado Jelly Bean. No entanto, desde o Android 4.3 KitKat, o Google tentou adaptar o Chrome como seu navegador oficial em dispositivos móveis. Sua interface é composta por: 1. Barra de pesquisa: Neste espaço você escreve o endereço web da página que deseja acessar. 2. Botão Gerenciador de Janelas: Ao pressionar este botão, você terá acesso a todas as janelas que tiver aberto no navegador. 3. Botão de configuração: Será exibido um menu de configuração onde você verá as opções para abrir uma nova janela e ajustar as funções do navegador, entre muitas outras coisas. 4. Página inicial: Este botão leva você para a página inicial da web. Você pode definir a página de sua preferência. Pressione o botão Configurações , selecione a opção Configurações e toque na opção Definir página inicial. 5. Páginas salvas: Permite acessar as páginas que você salva no navegador para poder acessá-las posteriormente. 6. Favoritos: Permite acessar suas páginas favoritas. Além desses dois navegadores, você pode baixar outros navegadores da Google Play Store.
