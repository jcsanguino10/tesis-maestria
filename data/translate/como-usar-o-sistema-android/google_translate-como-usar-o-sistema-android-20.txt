Navegando entre guias no Chrome No aplicativo Google Chrome, você pode abrir várias guias no mesmo navegador e acessar mais de uma página da Web ao mesmo tempo. A maneira como funciona depende da versão do sistema operacional. Estes são os dois procedimentos possíveis: Opção 1: Etapa 1: No navegador, pressione o botão Janela. Você poderá reconhecê-lo porque é um quadrado com um número dentro, que indicará o número de abas abertas. Passo 2: A página da web será minimizada e você verá todas as abas abertas. Toque no botão de adição (+) para abrir uma nova guia. Opção 2: Etapa 1: No navegador, toque no botão Configurações. Você reconhece porque são três pontos na vertical. Passo 2: No menu suspenso, selecione a opção Tab. Uma nova guia será aberta em uma janela separada. Você também pode abrir uma guia anônima para navegar sem deixar um registro em seu histórico. Toque no botão Aplicativos recentes para navegar entre as janelas abertas do navegador.
