Termos de computador Aqui estão alguns dos termos de computador mais usados. Outros termos comuns são: Navegador Este é o navegador de internet. Entre os &quot;navegadores&quot; ou navegadores mais populares estão: Chrome, FireFox, Safari e Microsoft Edge. Motor de busca na web Em inglês é conhecido como &quot;motor de busca&quot; e é um sistema de computador que funciona através de comandos para encontrar um elemento na web. Essas ordens são dadas por você, quando você insere palavras, imagens, etc. Alguns motores de busca na web são: Google, Bing e Yahoo! Procurar. Menu É o nome dado ao conjunto de opções oferecidas por um programa ou aplicativo. Por exemplo, a lista de opções que você tem no seu WhatsApp. Cache Memória temporária na qual os arquivos do computador são armazenados temporariamente. Caixa de diálogo Uma mensagem que aparece na tela de um dispositivo cada vez que um comando é executado. Banner Este é um tipo de publicidade que é feita na Internet. Caracteriza-se por aparecer em um espaço específico no site. Link Em espanhol, link . É a conexão de uma página web com outra.
