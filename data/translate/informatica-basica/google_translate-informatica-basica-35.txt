Sistemas operacionais para o computador O sistema operacional já está instalado no computador e a maioria das pessoas não faz alterações nele. No entanto, é possível atualizá-lo ou alterá-lo. Todos os sistemas operacionais utilizam uma interface gráfica de usuário, ou seja, o que permite usar o mouse para clicar em ícones e botões, ou interagir com qualquer outro elemento para executar ações ou tarefas. É assim que dizemos ao computador o que fazer. Os sistemas operacionais mais comuns para computadores, ou aqueles que serão oferecidos no mercado quando você estiver procurando por um computador, são: Microsoft Windows, Mac OS e Linux. Microsoft Windows Foi desenvolvido nos anos oitenta. Suas versões mais recentes são Windows 11, Windows 10; Windows 8, criado no ano de 2012; Windows 7, em 2009; e Windows Vista, criado em 2007. O Windows é instalado por padrão na maioria dos novos computadores. É por isso que ele é classificado como o sistema operacional mais popular. Mac OS X é o sistema operacional criado pela Apple Inc. e vem instalado em todos os seus computadores. As versões mais recentes são: Monterrey, BigSur e Catalina; e alguns, um pouco mais velhos, são: Mojave, HighSierra, Sierra. A Apple também oferece uma versão chamada MacOSServer, projetada para rodar em servidores. Linux Ubuntu é um sistema operacional de código aberto. Isso significa que ele pode ser modificado e distribuído por qualquer pessoa ao redor do mundo. Essa é uma de suas vantagens, pois você não precisa pagar por isso e pode escolher entre as diferentes versões que existem. Em computadores domésticos, o Linux, apesar de gratuito, é pouco utilizado, mas a maioria dos servidores e empresas utilizam o Linux por ser de fácil customização. As versões mais populares são Ubuntu, Debian, Linux Mint e Fedora; Encontre um que se adapte aos seus gostos e necessidades. A interface gráfica do usuário de cada sistema operacional parece diferente, portanto, alternar de um sistema para outro pode parecer estranho no início, mas não se preocupe, todos eles têm funções muito semelhantes. Todos os sistemas operacionais são projetados para serem fáceis de usar e os princípios básicos são os mesmos em qualquer um deles. Provavelmente, se você souber lidar com um, poderá se adaptar facilmente a outro. Antes da criação da interface gráfica do usuário (GUI), as pessoas usavam comandos no teclado para se comunicar com o computador.
