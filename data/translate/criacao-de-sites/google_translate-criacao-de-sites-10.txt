CSS - Cascading Style Sheets CSS é uma linguagem descritiva que permite definir a apresentação de um documento HTML. CSS não é uma linguagem de programação, é uma linguagem descritiva composta de regras que indicam como apresentar um documento html. Quando uma folha de estilo é associada a um documento html, o navegador interpreta tais regras e as aplica ao documento, modificando sua aparência. Regras Uma regra CSS é composta por um ou mais seletores seguidos por uma declaração que consiste em um ou mais pares de propriedade e valor. Um arquivo css pode conter várias regras. Seletores Os seletores indicam a quais elementos do documento uma regra deve ser aplicada. Uma regra pode conter um ou mais seletores, separados por vírgulas. Propriedades Propriedades são características visuais que os elementos html possuem. Uma declaração pode conter uma ou mais propriedades, cada uma com seu próprio valor e separadas por ponto e vírgula. Tipos de seletores Seletor por tag Este seletor corresponde aos nomes de tags dos elementos html, por exemplo p , a , h1 ou div . Isso significa que a regra será aplicada a todos os elementos desse tipo que aparecem no documento. Por exemplo, se queremos que todos os parágrafos tenham um certo alinhamento ou que todos os links tenham uma certa cor. Seletor por id Elementos HTML podem ter um atributo chamado id que deve ser único e serve para identificá-lo dentro do documento. O seletor deve começar com o símbolo # seguido do id do elemento e a regra será aplicada apenas a esse elemento. É útil estilizar os elementos que queremos destacar. Seletor por classe Os elementos HTML podem ter um atributo chamado class , que é usado para agrupar diferentes elementos sob o mesmo identificador. O seletor deve iniciar com o símbolo ( . ) seguido do nome da classe e a regra será aplicada a todos os elementos que o possuam. Seletor por estado Alguns elementos html podem ter estados diferentes, um botão pode estar desabilitado, um link já pode ter sido visitado ou o ponteiro do mouse pode estar sobre algum texto. O seletor de estado permite aplicar regras diferentes dependendo do estado dos elementos. Explorar X Selector por atributo Os elementos HTML possuem vários atributos, é possível aplicar regras aos elementos de acordo com o valor particular de seus atributos. Este seletor é indicado entre os símbolos [] especificando o nome e o valor do atributo. Ele pode ir como único seletor na regra ou depois de outro seletor. No exemplo aplicamos a regra a todas as imagens cujo arquivo é logo.png. Outros aparecem primeiro ou mais tarde em relação aos outros. Seletores de relação são usados para selecionar elementos com base em seu relacionamento entre si. Propriedades mais comuns O número de propriedades dos elementos html é extenso, você pode consultar a referência completa em sites especializados, abaixo mencionamos apenas algumas para você ter uma ideia. Propriedades de formatação de texto text-align : Permite alinhar textos à esquerda, direita, centralizado ou justificado. font-weight : Permite ajustar o peso (peso) da fonte. font-size : Permite ajustar o tamanho da fonte. font-family : permite escolher a fonte de um texto. Propriedades de posição superior, esquerda, direita, inferior: indicam, respectivamente, a distância do elemento das bordas do documento ou de elementos próximos a ele. position : indica como é calculada a posição de um elemento no documento, pode ser absoluta quando calculada em relação ao elemento que o contém ou relativa quando calculada em relação à sua posição original. float : indica como um elemento se comporta se houver espaço livre no documento, pode ser direito ou esquerdo indicando se o elemento ocupa os espaços livres à direita ou esquerda respectivamente. Espaçamento Propriedades de preenchimento : Indica o espaço livre entre um elemento e seus elementos internos. margin : Indica o espaço livre entre um elemento e elementos externos. Propriedades de Cor color : Indica a cor do elemento. background-color : Indica a cor de fundo do elemento. Efeitos de cor de borda, estilo de borda, largura de borda, raio de borda: indicam se um elemento tem borda ou não e quais são suas propriedades. box-shadow : Aplica um efeito de sombra a um elemento. Design responsivo Atualmente é possível aplicar diferentes regras dependendo das características do dispositivo onde o documento é exibido, isso é conhecido como design responsivo. Isso é possível por meio de consultas de mídia que são condições nas folhas de estilo que, quando cumpridas, permitem que certas regras sejam aplicadas. Uma consulta de mídia começa com o símbolo @ seguido da palavra mídia , depois o tipo de dispositivo e as condições entre parênteses () . Revise os comentários de código no exemplo a seguir.
