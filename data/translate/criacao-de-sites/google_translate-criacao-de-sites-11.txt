JavaScript JavaScript é uma linguagem de programação, o que significa que possui variáveis, funções, loops e condicionais entre outros recursos. Através dele podemos modificar os elementos html do documento, processar e validar dados do usuário ou fazer solicitações de recursos ao servidor. Conceitos básicos Variáveis Na programação, uma variável é um nome pelo qual podemos nos referir a um espaço na memória do computador. Em JavaScript não é necessário especificar o tipo de dado que a variável irá conter e uma variável pode conter qualquer tipo de dado a qualquer momento. Abaixo estão várias maneiras de declarar variáveis. X Condicionais Uma condicional é uma instrução de programação que permite que um ou outro código seja executado dependendo se o resultado de uma operação é falso ou verdadeiro. Em JavaScript temos comparações para igualdade, maior que e menor que, e operadores lógicos como e e ou. Olhe para o exemplo. X Ciclos Na programação é muito comum ter que aplicar os mesmos processos a uma grande quantidade de dados, para isso usamos ciclos, que nos permitem repetir a execução do mesmo código. Abaixo estão dois exemplos, o loop for e o loop while. Funções X O propósito das funções na programação é agrupar o código que é usado com muita frequência. Em JavaScript, as funções também podem se comportar como variáveis e essa é uma de suas características mais importantes. Confira os comentários no exemplo abaixo. X Objects A programação orientada a objetos nos permite pensar em um programa de computador como um conjunto de coisas que fazem alguma coisa. Na prática, um objeto se parece com uma estrutura de dados que também possui funções que operam em seus próprios dados. Sendo JavaScript uma linguagem de programação orientada a objetos, você pode ver várias operações com objetos no exemplo a seguir. Eventos A programação orientada a eventos nos permite escrever programas que não executam linearmente, mas executam parte do código em resposta a um evento. Um evento pode ser algo que acontece, por exemplo, quando você clica com o mouse ou a abertura de uma conexão de rede. JavaScript é uma linguagem orientada a eventos, na qual podemos conectar funções com eventos, a fim de interagir com o usuário. No exemplo, conectamos um evento de mouse com uma função anônima. X JavaScript no navegador Os navegadores oferecem um ambiente de execução de JavaScript através do qual podemos manipular o documento html e realizar algumas operações de rede. As possibilidades que temos são determinadas pela API DOM (Document Object Model - Application Program Interface) que é, simplificando um pouco, uma lista de objetos e métodos disponíveis para isso. Existem muitas operações que podemos fazer em um documento html, abaixo Mencionamos alguns. Operações de seleção Semelhante aos seletores css, temos operações que retornam um ou mais elementos do documento. Confira os comentários no exemplo abaixo. Operações X Style Através delas podemos alterar as propriedades dos elementos html do código JavaScript, como o estilo da fonte no exemplo a seguir. X Operações com elementos de mídia Elementos de mídia em particular possuem atributos e eventos especiais que outros elementos não possuem. No exemplo vemos alguns atributos do elemento video. X Operações de rede É possível fazer solicitações ao mesmo servidor que estamos visitando para obter recursos como texto, html, dados ou arquivos, isso é feito com a técnica AJAX (Asynchronous JavaScript And XML) que nos permite fazer a solicitação sem sair do documento atual. No exemplo, vemos uma solicitação GET. X Operações com eventos Através do JavaScript é possível associar funções a eventos do documento html, desta forma podemos gerar maior interação com os usuários.
