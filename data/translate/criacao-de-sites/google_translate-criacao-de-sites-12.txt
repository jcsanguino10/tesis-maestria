Modelos e frameworks web Para o desenvolvimento web nem sempre é necessário começar do zero, podemos construir em bases que encontramos livremente na rede. Nas páginas anteriores, apresentamos as três principais tecnologias da web: HTML, CSS e JavaScript. Cada um tem sua própria complexidade e ser capaz de dominá-los leva tempo, felizmente graças ao espírito aberto da web podemos ter bases para iniciar um desenvolvimento. Bibliotecas Em software, uma biblioteca é um componente reutilizável que facilita determinadas tarefas de programação. Dependendo da tecnologia, pode ser um conjunto de funções ou objetos. Ao integrar uma biblioteca ao nosso projeto teremos acesso a funcionalidades que levariam muito tempo para fazer e que foram bem desenhadas e testadas na biblioteca. Templates No contexto de desenvolvimento web, um template ( template ) refere-se a uma estrutura de arquivos estabelecida para um site, na qual é necessário apenas modificar e adaptar determinados dados. Frameworks Um framework vai um pouco além, embora possa oferecer opções semelhantes a bibliotecas ou templates, seu diferencial está no fato de propor um modelo, ou seja, uma forma particular de desenvolver software. Por esta razão, um framework é acompanhado de conceitos que devem ser bem compreendidos para poder utilizá-lo. Frontend refere-se aos componentes visíveis ao usuário: páginas, estilos e scripts. Backend refere-se aos não visíveis, como bancos de dados e servidores da web. Na prática, não é tão fácil distinguir entre bibliotecas, templates e frameworks, por isso é importante consultar a documentação e ter certeza do que eles nos oferecem e quais são as necessidades do nosso projeto. Abaixo mencionamos alguns disponíveis gratuitamente na rede. jQuery jQuery é uma biblioteca JavaScript que facilita a manipulação de um documento html. Existem duas bibliotecas adicionais que são jQuery UI para componentes visuais e jQuery mobile focada em smartphones. Materialize Materialize é um framework de desenvolvimento web baseado no conceito de Material Design do Google. Inclui componentes visuais, animações e até paletas de cores. Bootstrap Bootstrap é um framework para desenvolvimento web orientado ao design responsivo. Foi desenvolvido dentro do Tweeter e posteriormente lançado como código aberto. Angular AngularJS é um framework do Google que estende elementos html para facilitar o desenvolvimento de aplicações web dinâmicas. Backbone BackboneJS é um framework focado em sincronizar mudanças de dados com a forma como são apresentadas e com sua entrega ao servidor no caso de um site dinâmico. Design responsivo Amazium e Gridism são frameworks css para design responsivo, eles fornecem uma estrutura html e estilos predefinidos para projetar um site sob um modelo de grade. Modelos HTML Existem vários sites que oferecem modelos html gratuitos, Tooplate oferece uma coleção de modelos de sites gratuitos para download, bem como HTMLTemplates entre muitos outros. Para usar um framework ou biblioteca de front-end, basta baixar seus arquivos em seu projeto e referenciá-los nas páginas por meio de tags de link para folhas de estilo e scripts para código JavaScript.
