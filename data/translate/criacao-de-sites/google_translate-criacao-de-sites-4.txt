Conceitos da Web Muitos dos conceitos técnicos da Web são palavras em inglês. Outras são palavras comuns, mas com significado diferente no contexto tecnológico. Por exemplo, quando se fala em servidor, pode significar várias coisas. O servidor pode ser um computador de alto desempenho no qual o site está hospedado, mas também pode se referir a software de servidor, ou seja, software projetado para receber solicitações de outros computadores e dispositivos. Esses computadores que fazem solicitações a um servidor são chamados de clientes. Os clientes fazem solicitações ao servidor por meio do navegador. O navegador ou navegador da web é um aplicativo que nos permite entrar em contato com o servidor, receber sua resposta e visualizá-la. Essas solicitações que nomeamos são feitas usando um protocolo, que é uma definição dos passos necessários entre duas máquinas para estabelecer uma comunicação. Hypertext Transfer Protocol, ou HTTP, é o protocolo usado na web. As solicitações podem ter respostas diferentes. Por exemplo, a resposta pode ser uma página web, uma folha de estilo ou um script, entre outros. Uma página web é um documento de texto em linguagem HTML, uma folha de estilo é um documento de texto em linguagem CSS e um script é um documento de texto em linguagem JavaScript.
