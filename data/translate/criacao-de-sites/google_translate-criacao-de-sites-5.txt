Linguagens da Web Como vimos anteriormente, um servidor da Web pode retornar uma página da Web, uma folha de estilo ou um script, entre outros tipos de arquivos, como resposta. Para criá-los, são utilizadas linguagens da web: HTML, CSS e JavaScript, que são três das linguagens mais importantes. Logos de W3C e pixelbazar. Agora vamos ver do que se trata cada um deles: HTML - HyperText Markup Language A HyperText Markup Language é a linguagem usada para escrever uma página da web. É composto por tags que marcam o início e o fim de uma estrutura como um parágrafo de texto, uma imagem ou um link. HTML não é uma linguagem de programação, pois não possui recursos como sequências, condicionais e loops. Arquivos escritos nesta linguagem são informalmente chamados de páginas. JS - JavaScript JavaScript é uma linguagem de programação. É a linguagem que permite programar determinadas funcionalidades no navegador, como animações ou validação de dados. Você ouvirá arquivos escritos neste idioma como scripts nomeados informalmente. CSS - Cascading Style Sheet Folhas de estilo em cascata são uma linguagem que define a aparência de uma página web. CSS não é uma linguagem de programação, é mais uma linguagem descritiva. Com CSS, são definidas as propriedades visuais do conteúdo de uma página: cores, tipo e espessura da fonte, tamanho dos elementos do conteúdo, etc. Eles são frequentemente chamados informalmente como os estilos.
