Linguagens de programação web Para o desenvolvimento no backend web podemos escolher entre várias linguagens de programação, vamos ver algumas delas. Apresentamos uma visão geral das possibilidades de desenvolvimento em diferentes linguagens, a partir daqui você pode se aprofundar visitando os manuais técnicos e tutoriais compilados no final desta lição. Todas essas linguagens/plataformas nos permitem acessar parâmetros de solicitação, podemos escrever arquivos binários ou texto html na resposta, podemos gerenciar sessões e nos conectar a vários bancos de dados. Por outro lado, cada plataforma possui frameworks diferentes que oferecem formas de modelar nosso desenvolvimento, utilidades comuns e sistemas de templates. PHP PHP é uma das linguagens mais reconhecidas no desenvolvimento web, foi criada em 1994 por Rasmus Lerdorf para sua página pessoal, posteriormente foi lançada como open source em 1995 e a partir daí teve uma longa evolução até o que é hoje. PHP é principalmente voltado para o desenvolvimento web. PHP é uma linguagem multiparadigma, ou seja, você pode programar de forma procedural (funções e variáveis) ou orientada a objetos (objetos, atributos e métodos). O PHP deve estar integrado a um servidor web como Apache ou NginX, entre outros. Dentre alguns de seus frameworks temos: Laravel, Symfony, Zend 2 e CakePHP. Explore X Java Java começou a ser concebida em 1991 por James Gosling e foi finalmente lançada ao público em 1995. É uma linguagem orientada a objetos, portátil para diferentes arquiteturas, que permite rodar desde telefones celulares e computadores pessoais até servidores e supercomputadores. Java oferece duas tecnologias para desenvolvimento web: Servlets e páginas JSP (Java Server Pages). Servlets são objetos que possuem métodos que respondem a requisições da web, o desenvolvedor cria subclasses desses objetos para implementar a lógica de seu site dinâmico ali. Por sua vez, JSP permite misturar código Java com tags html. Para rodar Servlets ou páginas JSP, você precisa de um servidor web especializado em tecnologia Java como WildFly, Tomcat ou Glassfish. Entre seus frameworks mais conhecidos estão: Spring, Wicket, Struts, Play! e JSF. Explore X ASP.NET ASP é a tecnologia da Microsoft para desenvolvimento de sites dinâmicos, sua versão mais recente, ASP.NET, foi lançada ao público em 2002. Utiliza o ambiente de tempo de execução .NET para que seja possível programar em linguagens suportadas para isso como C# ou Visual Basic. É possível misturar código de programação com tags html ou separar código e tags em arquivos diferentes. Um servidor Microsoft IIS é necessário para executar páginas ASP.NET. Entre seus frameworks encontramos: ASP.NET MVC, ASP.NET WebAPI e NancyFX. Explore X Com o uso de frameworks e sistemas de template, evita-se misturar o código de programação com tags html, o que melhora a manutenibilidade de grandes projetos. Equivalentes É possível fazer desenvolvimento web em praticamente qualquer linguagem, você pode encontrar opções em Ruby, Perl, Python, C++, JavaScript, Haxe... . Podemos destacar Node.js para desenvolvimento backend JavaScript, cppCMS para desenvolvimento C++, Ruby On Rails para desenvolvimento Ruby e Django para desenvolvimento Python. Para escolher entre uma tecnologia ou outra, você deve levar em consideração vários fatores: o tamanho do projeto, as linguagens em que você tem experiência, desempenho, os frameworks disponíveis, a qualidade da documentação e estabilidade, entre outros.
