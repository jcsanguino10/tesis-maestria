HTML - Hyper Text Markup Language HTML é uma linguagem que descreve um documento de hipertexto. HTML não é uma linguagem de programação, é uma linguagem descritiva composta por tags, um texto que marca o início e o fim de um elemento html. As tags são interpretadas pelo navegador e o resultado é o que vemos na tela de nossos dispositivos. Tags básicas Abaixo você verá as tags básicas que todo documento html deve ter: html A tag html é a primeira tag que aparece em um documento, ela marca precisamente o início e o fim de todo o documento. head Na tag head descrevemos as coisas que são comuns ao documento como o título, folhas de estilo e scripts. body Dentro desta tag escrevemos o conteúdo do documento como tal. Tags de cabeçalho Essas tags devem ficar dentro da tag de cabeçalho. Eles descrevem metadados e recursos comuns a todo o documento. title Dentro da title tag devemos escrever o título do documento, que ficará visível na aba do navegador. meta A metatag é usada para descrever informações sobre o documento, como o autor, uma breve descrição e algumas palavras-chave. Essas informações são importantes para que os mecanismos de busca e as redes sociais saibam do que se trata a página. link Com a tag link podemos nos referir a um recurso externo, seu uso mais comum é associar uma ou mais folhas de estilo a um documento HTML. script A tag script é usada para fazer referência a arquivos de código externos ou você também pode incluir código diretamente nele. Desta forma associamos o código JavaScript a documentos HTML. Tags estruturais Essas tags não possuem uma exibição específica, mas são úteis para estruturar o documento. div O objetivo da tag div é agrupar diversos elementos dentro de um único bloco, isso é útil por exemplo ao aplicar estilos css, pois podemos aplicar um estilo a todos os elementos que estão dentro do mesmo bloco div. span Com a tag span também é possível agrupar elementos, mas não dentro de um bloco, mas de forma linear. header A tag header nos permite definir o cabeçalho do documento, geralmente lá encontramos o logotipo e o menu de um site. footer Com a marca de rodapé definimos o rodapé do documento, geralmente aqui temos os links para informações corporativas, perguntas frequentes e informações de contato. Tags de interface do usuário Essas tags apresentam controles visuais no documento, nós as utilizamos para receber informações do usuário, por exemplo em formulários. select A tag select apresenta uma lista suspensa de opções. Você pode clicar nas setas na parte de visualização para testar seu funcionamento. Botão Explore X Como o próprio nome indica, esta tag apresenta um botão. Caixa de seleção Explorar X Esta tag apresenta um botão de tipo de verificação. Esses tipos de botões geralmente são usados em grupos para exibir várias opções. Botão de opção Explore X Esta tag apresenta um botão de opção. Esse tipo de botão geralmente é usado em grupos para escolher uma única opção entre várias. Explorar entrada X Esta tag é utilizada para apresentar ao usuário um campo de entrada de dados que pode ser texto, numérico, data ou outros tipos. Você pode escrever dentro do campo para verificar seu funcionamento. Explorar a área de texto X Esta tag apresenta uma área que permite a entrada de uma quantidade maior de texto. Você pode escrever um parágrafo dentro do campo para verificar seu funcionamento. Explorar X Tags multimídia Essas tags nos permitem incluir outros elementos além do texto. Alguns aparecem a partir da versão 5 do padrão HTML e possibilitam a reprodução de áudio, vídeo e elementos 2D e 3D sem o uso de plugins adicionais. img Através da tag img podemos incluir imagens estáticas ou animadas em nosso documento html. Explore o áudio X Com a tag audio podemos incluir arquivos de áudio no documento e reproduzi-los diretamente pelo navegador ou navegador. Explore X video Com a tag video podemos incluir vídeos no documento, com a possibilidade de exibir controles de reprodução, como pausar, avançar e visualizar em tela cheia. Explore X canvas A tag canvas permite incluir gráficos personalizados em 2D ou 3D, é uma área do documento onde podemos desenhar através de código. Explorar tags de formato X Essas tags representam elementos de texto com características diferentes. Títulos Os títulos variam do mais importante (1) ao menos importante (6), com tags de h1 a h6. Explore X Paragraph Os parágrafos são enquadrados dentro da tag p, agrupando o texto que faz todo o sentido. Explore X Lists Temos dois tipos de listas: listas que são numeradas automaticamente com a tag ol e listas com itens não numerados com a tag ul. Explorar X Tags de hipertexto Essas tags permitem que você navegue de um documento para outro no modelo de leitura de hipertexto. Link A tag (a) representa os links para outros documentos html na web, são compostos pelo texto visível e pela url de destino.
